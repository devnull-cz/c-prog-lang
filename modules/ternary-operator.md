# Ternary operator

```
cond ? expr1 : expr 2
```

Is the same as:

```
if (cond) expr else expr2
```

Example:

```
max = (i > j) ? i : j;
```

Note that if we add a semicolon, an expression becomes a statement, e.g.:

```
(i == j) ? i++ : j++;
```

That will increment `i` or `j`.  The parentheses are not needed but generally
used for better readability.
